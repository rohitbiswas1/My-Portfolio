## 👨‍💻 About

This is my personal developer portfolio built with vanilla HTML, CSS, and JavaScript — no frameworks, no build tools, just clean and fast front-end code. It showcases my education, certifications, research interests, tech stack, and featured projects.

---

## ✨ Features

- **Dark / Light Mode** — persisted via `localStorage`
- **Smooth Scroll Navigation** — with active link highlighting
- **Certificate Image Modal** — click-to-expand with keyboard (`Esc`) and backdrop close support
- **Intersection Observer Animations** — fade-in on scroll for every section
- **Fully Responsive** — mobile-first layout across all screen sizes
- **SEO Ready** — meta tags, Open Graph, semantic HTML, and ARIA attributes
